Idlebreaker is to prevent computer windows from idle. Within 10 seconds, idlebreaker will break computer idle. In order idlebreaker to function properly, some procedures need to be set. For instance, allowing idlebreaker to be active automatically when Windows starts up:
1. Locate the appication folder in C:\ is better
2. Open Run by pressing Window + R, type "shell:startup" click "OK". You will be directed to Startup folder location.
3. Make idlebreaker.exe shortcut, and copy/move it to Startup folder.
4. Finish. You can check idlebreaker will appears automatically in Startup in Task Manager.
Note: 1. Windows 7 has different method to set application in Startup. 2. Idlebreaker seems not taking effect when computer screen is mainly directed to Task Manager.
